== Description ==

**WP-CopyRightPro** is a plug-in that prevents the copying of texts and images from your blog,
if you install this plug-in, your content of wordpress will be protected.

= This plug-in will =

* **Disable selection of text**
* **Disable right click on your Wordpress**
* **WP-CopyRightPro doesn't have problems with search engines**


**[Download now](http://downloads.wordpress.org/plugin/wp-copyrightpro.zip)**


== Installation ==

This section describes how to install the plugin and get it working.

1.Download the file **WP-CopyRightPro.zip**

2.Unzip WP-CopyRightPro and upload the folder to the `/wp-content/plugins/` directory.

3.Enter in your admin panel of wordpress and activate the plugin.

**Now your content of wordpress will be protected**
